## Real timeframe

I wasn't all that careful with git on this one, and squashing the history
at release didn't help — so the git dates don't tell the whole story. The
actual timeline shows up in the Maildir archive under `.messages/`
instead: the filenames follow the standard Maildir
`<unix-timestamp>.<id>.<host>` convention, and the archived messages span
2025-11-03 to 2025-11-05 (UTC) — that's the real multi-model collaboration
the `msg` tool and `mm.py` were built to carry. So this project is really
about eight months older than the public-release date makes it look.
